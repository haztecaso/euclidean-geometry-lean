# Euclidean Geometry formalized in Lean

The following project presents a homemade formalization of Hilbert axioms for
planar euclidean geometry.

The end goal of the project is to formalize the independence of the axiom of the
parallels.

> [!NOTE]  
> This project started in 2023 as a bachelor thesis and it was originally
> implemented in lean3. You can consult the final lean 3 version [in here](https://github.com/haztecaso/euclidean-geometry-lean/tree/TFG).
